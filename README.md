# Student-Faculty Management System

A full-stack Student-Faculty Management System developed using React.js, Spring Boot, PostgreSQL, and JWT Authentication. The application provides secure role-based access control for Super Admin, Admin, Faculty, and Students.

## Features

- User Authentication and Authorization using JWT
- Role-Based Access Control
- Student Management
- Faculty Management
- Course Management
- Attendance Management
- Secure REST APIs using Spring Security
- Responsive User Interface using Bootstrap
- CRUD Operations
- Profile Management
- Cloud Database Integration using Supabase
- Frontend Deployment using Vercel
- Backend Deployment using Render

## Tech Stack

### Frontend
- React.js
- Bootstrap
- Axios
- React Router

### Backend
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate
- JWT Authentication
- Maven

### Database
- PostgreSQL (Supabase)

## Security Features

- JWT Token Authentication
- BCrypt Password Encryption
- Protected Routes
- Secure REST APIs
- Role-Based Authorization

## Deployment

- Frontend: Vercel
- Backend: Render
- Database: Supabase

## Project Architecture

The project follows a layered architecture:

- Controller Layer
- Service Layer
- Repository Layer
- DTO Layer
- Entity Layer
- Security Layer

## Objectives

The main objective of this project is to simplify educational institution management by providing a centralized system for managing students, faculty, courses, attendance, and user authentication securely.
